Railway 환경변수 설정 - 복사해서 사용하세요
═══════════════════════════════════════════════════════════

🔴 필수 설정 (반드시 입력)
─────────────────────────────────────────────────────────

# OpenAI & Assistant
OPENAI_API_KEY=your_openai_api_key_here
CAIA_ASSISTANT_ID=asst_your_assistant_id
CAIA_THREAD_ID=thread_your_thread_id

# Telegram
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
TELEGRAM_CHAT_ID=your_telegram_chat_id

# DB증권 API (K200 선물 감시)
DB_APP_KEY=your_db_app_key
DB_APP_SECRET=your_db_app_secret

─────────────────────────────────────────────────────────

🟢 권장 설정 (기본값 있음, 복사 권장)
─────────────────────────────────────────────────────────

# Sentinel 설정
SENTINEL_BASE_URL=https://fastapi-sentinel-production.up.railway.app
SENTINEL_KEY=your_sentinel_secret_key
LOG_LEVEL=INFO

# DB증권 K200 선물 모니터링
DBSEC_ENABLE=true
DBSEC_USE_REST=true
DB_POLL_INTERVAL_SEC=180

# Market Watcher 설정
WATCH_INTERVAL_SEC=180
VIX_FILTER_THRESHOLD=0.6
FORCE_MARKET_OPEN=true

# 알림 임계값
DB_ALERT_THRESHOLD=1.5
DB_WARN_THRESHOLD=0.8

─────────────────────────────────────────────────────────

🔵 선택 설정 (필요시)
─────────────────────────────────────────────────────────

# Hub 연동 (선택)
HUB_URL=your_hub_url
CONNECTOR_SECRET=your_connector_secret

# Caia 상세 로그 (디버깅용)
CAIA_VERBOSE=0

# 타임아웃 설정
CONNECT_TIMEOUT=10
READ_TIMEOUT=60
RUN_POLL_MAX_WAIT=90

# 중복 알림 방지
DEDUP_WINDOW_MIN=30

─────────────────────────────────────────────────────────

📝 Railway 설정 방법:
1. Railway 대시보드 접속
2. 프로젝트 선택 > Variables 탭
3. 위의 환경변수들을 복사해서 입력
4. Deploy 버튼 클릭

⚠️ 주의사항:
- your_xxx_here 부분을 실제 값으로 교체하세요
- 민감한 정보는 절대 GitHub에 커밋하지 마세요
- 토큰/키 값에는 공백이 없어야 합니다

✅ 설정 후 확인:
curl https://your-project.railway.app/health
curl https://your-project.railway.app/sentinel/dbsec/health
